Brainstorm

This tool was created to help game developers rapdily introduce a simple multi-brain BCI sytem on the web. Brainstorm has been written for integration into a BCI processing pipeline using Brainflow-compatible headsets to synchronize events occuring in the game with specific brain activity.

Getting Started

To get started, you simply need to use our API in your Javascript code. This includes all the necessary back-end components required for including data from multiple brains into your project.
